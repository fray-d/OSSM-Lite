6. Setup WiFi for over-the-air updates

WiFi? I thought this thing connected to devices over bluetooth? It does, but if you want to get updates without coming back here (and overwriting all those configurations you just set up) then you should probably enable over-the-air updates.

On the control board there are two buttons. One near the LED restarts the device, the other is a function button. Click and hold this second button.

If the LED flashes yellow then your device just started hosting a wifi network with a name like "OSSM Setup", or whatever you renamed your device to. Connect to it to complete configuration! You can figure this out, I have faith in you.

If it blinks orange then you were already connected to wifi, and congratulations you're starting to pull down the latest firmware update. Click and hold that button whenever you want to check for a new update. The device does not check versions... it just updates to whatever it finds.

You can also trigger either of these functions with the wired remote. If you read the previous pages you probably knew what those colors meant...and maybe that this button existed!
