OSSM Web Flasher

min-max-stroke

Flash the latest min-max-stroke firmware directly from your browser. No build tools required — just a USB-C cable and Chrome or Edge.

This branch adds set:min and set:max BLE commands, replacing the depth/stroke model with explicit min/max position control.

Requirements: Chrome or Edge on desktop (not mobile). Windows may need a CP210x USB driver.

Source: aperturetest/OSSM-hardware @ min-max-stroke
