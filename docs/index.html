OSSM Lite

Documentation

The OSSM Lite firmware is intended to be both easy to use and empower the user to customize their experience. Read this documentation in order to commision a new device, or jump to the section you need with the links below. You are encouraged to read the entire document before seeking support elsewhere. It's not terribly long and has most of the answers.

Steps to set up this OSSM firmware and it's features:

1.   Flash the firmware
2.   Review the blink code information
3.   Set motor constraints
4.   Choose a homing method
5.   Configure rail direction
6.   Name your device
7.   Setup WiFi for over-the-air updates
8.   Learn about play modes
9.   Advanced Penetration Preset Tool
10.   Funscript Player
