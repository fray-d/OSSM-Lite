10. Advanced Penetration Preset Tool

With this tool you can manage your presets. It is not meant for play or authoring new presets as it does not expose all of the settings in a user friendly way. A minimum set of controls are offered.

Speed and depth are available so you can view a preset while you choose a name. Set the active preset by selecting it from the dropdown. Setting speed will not work if you have a wired controller attached.

Renaming is possible through a three step process:
  • Pick the old pattern from the dropdown setting it as active.
  • Type the new name in the box and save it with those existing settings.
  • Reselect the old pattern by name and click delete.

Factory reset will delete all presets and load the default library of presets.

Requirements: Chrome, Edge, or Opera on desktop (not mobile).
