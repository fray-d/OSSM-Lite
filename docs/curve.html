6. Define speed curve

Different people have different priorities when it comes to the speed of the OSSM. Some are looking for the fastest device they can possibly get, others would prioritize granular control of low-end speed. This setting adjusts the ramp-up curve that governs the speed setting of the OSSM. All settings will allow for max speed based on motor RPM. If you wish to limit max speed see the motor parameters configuration.

Requirements: Chrome, Edge, or Opera on desktop (not mobile).
