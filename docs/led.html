OSSM Lite

2. Review blink code information

I know what you're thinking... Really, next step is to learn about colors?

Not every OSSM has the same remote connected, but they all have the status LED. It can give you clear indication of what is going on with your device. If this is your first boot it's probably already blinking angrily at you right now. The list below is in the order you are most likely to experience them.

Color Pattern Meaning
Purple Slowly Pulsing Device is attempting to home
Red Rapid Flashing Device is in an error state. Likely this means homing has failed. Check your power cable is connected and the belt tension. Then restart.
Dark Blue Slowly Pulsing Device is idle and waiting for a remote to connect over bluetooth
Dark Blue (Dim) Slowly Pulsing Same as above, but it's been a while...
Cyan Slowly Pulsing Device is in a play mode using only the wired remote.
Green (Dim) Slowly Pulsing Wireless remote connected
Green Quick Flash Wireless communication sent/received
Yellow Slowly Pulsing WiFi setup enabld. See wifi section of this documentation
Orange Rapid Flashing Device is pulling down an update. Do not power off. Will restart on it's own.
